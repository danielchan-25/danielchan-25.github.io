Hugo如何使用?

02 Apr, 2024

作为一个经常逛技术论坛、博客的人，很羡慕大神们都有一个非常炫酷的主页，我也很想搭建一个博客。同时作为一位后端程序员，我对前端技术只限于HTML，查阅了很多资料，终于找到一个非常适合小白入门的博客，那就是 Hugo

1. Hugo 是什么？

主页：https://gohugo.io/

Github：https://github.com/gohugoio/hugo

Hugo 是最流行的开源静态站点生成器之一。 凭借其惊人的速度和灵活性，Hugo 让构建网站再次变得有趣。

2. 如何使用 Hugo ？

2.1 下载安装 Hugo

下载地址：https://github.com/gohugoio/hugo/releases

将源码包下载完成后，放置至任意路径，随后将该目录添加至环境变量即可。

2.2 新建一个网站

E:\>hugo new site mywebsite
Congratulations! Your new Hugo site was created in E:\GitHub\mywebsite.

Just a few more steps...

1. Change the current directory to E:\GitHub\mywebsite.
2. Create or install a theme:
   - Create a new theme with the command "hugo new theme <THEMENAME>"
   - Install a theme from https://themes.gohugo.io/
3. Edit hugo.toml, setting the "theme" property to the theme name.
4. Create new content with the command "hugo new content <SECTIONNAME>\<FILENAME>.<FORMAT>".
5. Start the embedded web server with the command "hugo server --buildDrafts".

See documentation at https://gohugo.io/.

2.3 将喜好的主题安装至网站

我使用的主题：https://github.com/janraasch/hugo-bearblog

  1. 下载源码包，放置：mywebsite\themes\ 下
  2. 修改 hugo.toml 文件：theme = ['hugo-bearblog']

2.4 新建一个网页

使用 Hugo 命令建立一个网页：

E:\>hugo new blog/first_post.md

2.5 运行 Hugo 服务器

E:\>hugo	# 构建网站
E:\>hugo server	# 运行 Hugo 服务器

此时浏览器中访问：http://localhost:1313 ，即可浏览网站并进行调试。

3.如何把 Blog 推送至 Github ？

为什么要推送给 Github，因为 Github 能免费帮你托管网站，无需花精力维护服务器、网站等，那应该怎么推送上去？

  1. 首先你要有 Github 账户，并新建立一个仓库，名称为：[你的Github名称].github.io

  2. 回到本地，mysite\public 目录下，是把 public 中的文件推送至 Github。

    先初始化仓库，并建立远程连接，然后推送上去：

    git init
git remote add origin https://github.com/danielchan-25/danielchan-25.github.io.git

  3. 回到 Github ，等待 Action ，建立成功后就可以访问网站了。

如果访问网站时，显示页面为404，那应该修改 hugo.toml 文件，将 baseURL 修改为你的网站 URL。
